HTML5 Explained

Description

HTML5 (Hyper Text Markup Language) is the latest version of the HTML language which is used to display text on a web page. The concept to learning HTML is very simple and can take only 2 weeks to pick up the very basics. This language is the main display of text on any webpage.

What make is to easy?

What makes HTML so easy to learn, well, HTML is coded using tags such as head, title, body, footer and so on. These tags all display the basic format of a web page and once you know even a basic tags, you can start building your own website! The only downfall will be your website will look not so good and that is where we go onto CSS3...

Candidates

This is the most commonly used front end development language across the world as it is the easiest to learn and all candidates will put this on their CV. However, even though they say they can code in HTML and know the basics, does not make them an expert. I would recommend testing their knowledge and get them to show you an example of their code or even test them yourself with either your own found knowledge or by using the Technically Compatible test which can be sent to you through Hays.

Clients

There will be no limit to the clients which will want someone to know how to code HTML as mentioned above, it is the most used language in the world. Clients will also want to join their HTML knowledge with something called CSS which we will go over in the CSS3 section.

Golden Concept

The 'Golden Concept' to remember from the above will be to know, HTML is the most used web page language in the world but also the easiest for beginners & will be working together with CSS as the 2 main front end languages for beginners.
